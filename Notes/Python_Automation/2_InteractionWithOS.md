# Operating System
The software that manages everything that goes on in the computer. That means, it:
- Reads, writes and deletes files from the hard-drive.
- Handles how apps, programs and processes start, how they interact with each other and how they eventually finish. 
- Manages how memory gets allocated to different processes.
- Manages how network packets are sent and received.

There are two main parts to an operating system: 
 1.  Kernel : 
        - Main part of the system.
        - Talks directly to hardware and manages system's resources (disk, memory, network).
 2.  User space :
        - Everything outside the kernel. Basically all the things that a user can interact with directly like a program or something with a user interface. 

Some OS include Windows (Microsoft), MacOS (Apple but some of its kernel and user-space tools come from Unix's BSD), Linux (open-source but originally developed by Linus Torvalds) and Unix (Bell Labs).

### Linux 
- Heavily used in business infrastructure and most servers.
- Fundamental ideas of how it works are based on Unix principles.
- Since it is open-source, it has a lot of different versions (distributions) packaged by different organizations unlike Windows and MacOS. 
- Some distributions include Ubuntu, Debian, RedHat and ChromeOS. 
- The Android operating system (on phones) also runs a Linux kernel. 

## Why python?
Python is a cross-platform language. Since it is cross-platform, we can use the same Python code to get to our goal on any operating system whether the goal is opening files, processing text or managing running processes.
